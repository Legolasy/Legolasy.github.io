15.三数之和

给定一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a ，b ，c ，使得 a + b + c = 0 ？请找出所有和为 0 且 不重复 的三元组。

示例 1：

输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
示例 2：

输入：nums = []
输出：[]
示例 3：

输入：nums = [0]
输出：[]

提示：

0 <= nums.length <= 3000
-105 <= nums[i] <= 105

Solution&Key Points

1.三重循环 满足a≤b≤c

2.固定a 则二、三重循环实际为并列关系 因为对b’>b 有c’<c 双指针

3.每重循环 要保证a b c 和上一次枚举的数字不重复
